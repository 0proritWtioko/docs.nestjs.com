Async Components

Sometimes the application start has to be delayed until some asynchronous tasks will be finished, for example the connection with the database will be established. That's why Nest provides a way to create an async components.

To create an async component, we're using the async factory. 

{{ asyncComponent }}
Hint Read more about custom components here.

These asynchronous components might be simply injected by another components. Each component, which depends on the async components will be instantiated when the async component would be already resolved.
